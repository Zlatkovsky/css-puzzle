A fun CSS Puzzle!

This is a simple HTML page demonstrating a CSS puzzle. The root div has two classes applied to it: <div class="ui-framework-item container">

The ui-framework-item class has a simple rule: { padding: 10px }. Meanwhile, the container class overrides the padding for three of the sides: 

          padding-left: 150px;
          padding-right: 150px;
          padding-bottom: 30px;

When the dynamic content is loaded, it also defined a ui-framework-item class, but because they use the same pretend UI framework, it's the exact same rule: 

          { padding: 10px }

What happened?
